Quaternion Function Reference

qtfm_helpup

Rebuild QTFM HTML help files from XML masters

Syntax

qtfm_helpup

Description

qtfm_helpup processes the QTFM XML help files to produce the HTML files that are visible to the user in the MATLAB® help window. There is no need for the user to run this function (although it does no harm) unless some of the XML files have been updated from the CVS repository.
