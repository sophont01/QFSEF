Quaternion Function Reference

qtfm_helpdb

Build or rebuild QTFM searchable help database

Syntax

qtfm_helpdb

Description

qtfm_helpdb runs a MATLAB® function that constructs a searchable database from the QTFM help information. If the database already exists, it is replaced with a newly created version. These databases are specific to the version of MATLAB® that created them, so if you upgrade to a new version of MATLAB® you should re-run this function. The database is not distributed with the toolbox because of this specific link to the MATLAB® version that created it.
