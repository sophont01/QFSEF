Quaternion Function Reference

qtfm_version

Return QTFM version number

Syntax

V = qtfm_version

Description

qtfm_version reads the version number of the currently running QTFM code. It also checks Sourceforge (if an Internet connection is available) and outputs a warning if there is a more recent version available at Sourceforge, with the version number of the latest version. If no Internet connection is available, a warning is output.
